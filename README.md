# Simple Password Generator in python3

## Installation

```sh
git clone https://github.com/AlexRandomed/Simple-Password-Generator/
cd Simple-Password-Generator
chmod +x generate.py
pip3 install -r requirements.txt
```

## Usage

```sh
./generate.py 
or
python3 generate.py
```
